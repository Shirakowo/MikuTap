# Hatsune Miku - Tap Wonder Remake

The Remake version of 'Hatsune Miku - Tap Wonder' using Unity

Hatsune Miku - Tap Wonder (Tap Wonder or MikuTap) is a mobile game which develop by ATeam Entertainment Inc. (ATeam)\
However, ATeam stopped develop and support MikuTap in 25 Jan 2024. \
We are trying to reamke this game using Unity.

# Status
Now MikuTap Remake is still in developing, but we don't know when we will stop this project because maybe of some copyright or something else.
